eHealth Internet of Things Device

This device is a small passive device which will collect and display meaningful data on the context in which we decide to put it in. Just set it up in a few short minutes and let it go!

Once the device has been set up you can sign in and see realtime data that your device is measuring, or even set the device to perform specific actions!
